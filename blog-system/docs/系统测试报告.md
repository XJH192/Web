## 3. 测试中发现并修复的问题

### 3.1 AI 接口缺少登录校验

现象：未登录用户可以直接调用 AI 接口，可能消耗外部模型额度。

修复：五项 AI 接口统一要求有效登录令牌，独立 AI 页面同步传递令牌。

### 3.2 审核通过通知与需求冲突

现象：管理员通过文章或评论后，内容作者仍收到“审核通过”通知。

修复：删除两类冗余通过通知；保留公开后的真实互动通知和审核驳回通知。

### 3.3 全功能测试脚本位置不当

现象：PowerShell 文件放在主题 `scripts/` 目录时会被 Hexo 误当 JavaScript。

修复：测试脚本移动到 `blog-system/tests/`。

### 3.4 Windows PowerShell UTF-8 兼容

现象：Windows PowerShell 5.1 直接读取无 BOM 的中文脚本可能乱码。

修复：增加 ASCII 启动包装脚本，以 UTF-8 显式读取主测试脚本。
